# Hindi N-gram Language Model

This project implements a basic Natural Language Processing (NLP) language model using Hindi product review data.

## Project Overview
The goal of this project is to build a statistical language model that can predict word sequences using N-gram techniques.

## Techniques Used
- Unigram Model
- Bigram Model
- Laplace (Add-1) Smoothing
- Sentence Probability Calculation
- Perplexity Evaluation

## Dataset
Hindi Product Review Augmented Dataset.

## Libraries Used
- pandas
- nltk
- collections
- math

## Steps Implemented
1. Load Hindi review dataset
2. Preprocess text and tokenize words
3. Generate Unigrams and Bigrams
4. Calculate Bigram Probabilities
5. Apply Laplace Smoothing
6. Compute Sentence Probability
7. Evaluate Model using Perplexity

## Example Test Sentence
यह फोन बहुत अच्छा है
